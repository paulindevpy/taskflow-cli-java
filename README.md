# 🚀 TaskFlow CLI - Java

Sistema de gestão de tarefas desenvolvido em Java com foco em lógica de negócio, orientação a objetos e interação via terminal.

---

## 🧠 Sobre o projeto

Aplicação CLI (Command Line Interface) que permite gerenciar tarefas com funcionalidades completas de CRUD, incluindo validações e controle de estado.

---

## ⚙️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (OOP)
- ArrayList
- Scanner (entrada de dados)
- Tratamento de erros (try/catch)

---

## 🔥 Funcionalidades

- Criar tarefas
- Listar tarefas
- Atualizar status (PENDENTE / EM_ANDAMENTO / CONCLUÍDA)
- Remover tarefas
- Validação de dados
- Controle de ID automático

---

## 🧩 Arquitetura

O projeto foi estruturado com separação de responsabilidades:

- `Model` → dados da aplicação
- `Service` → lógica de negócio
